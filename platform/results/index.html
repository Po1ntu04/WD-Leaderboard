教师会话台

统一固定课堂常用命令、路径和提交模式。
1. 单个 pred.txt 评测
python platform/app/score_submission.py --submission path/to/pred.txt --name 学生提交名 --submission-group 课堂提交 --raw test_assets/platform_eval_v2_draft/raw.txt --gold test_assets/platform_eval_v2_draft/gold.txt --manifest test_assets/platform_eval_v2_draft/gold_manifest.csv

若写运行时间，请放在 pred.txt 最后一行：# runtime_seconds: 0.183

2. 可执行包评测
python platform/app/score_executable_submission.py --submission-dir path/to/submission_dir --name 学生提交名 --submission-group 课堂提交 --raw test_assets/platform_eval_v2_draft/raw.txt --gold test_assets/platform_eval_v2_draft/gold.txt --manifest test_assets/platform_eval_v2_draft/gold_manifest.csv
3. 批量会话运行
python platform/app/run_session.py --session-name class_session --prediction-dir path/to/predictions --executable-dir path/to/submissions --raw test_assets/platform_eval_v2_draft/raw.txt --gold test_assets/platform_eval_v2_draft/gold.txt --manifest test_assets/platform_eval_v2_draft/gold_manifest.csv
4. 本地展示服务
python -m http.server 8765 --directory platform/results
